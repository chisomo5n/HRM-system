This is a web-based Human Resource Management System (HRMS) built using HTML, CSS, PHP, and Java. The system helps organizations efficiently manage employee data, attendance, leave requests, and payroll. It is designed to streamline HR processes, reduce manual work, and improve accuracy.

Features

Employee registration and profile management

Attendance tracking and leave management

Payroll calculation and report generation

Secure login and role-based access

Technologies Used

Frontend: HTML, CSS

Backend: PHP, Java

Database: MySQL

Installation / Setup

Clone the repository:

git clone https://github.com/chisomo5n/HRM-system.git

Import the database file (if provided) into your local MySQL server.

Update the database connection settings in the PHP configuration files.

Open the project in your browser via a local server (e.g., XAMPP, WAMP).

Usage

Admins can add, update, and manage employee information.

Employees can check attendance and submit leave requests.

Payroll reports are generated automatically based on attendance and leave data.
